# 制作方式

<h2 id="lunch-1">🍅 番茄鸡胸肉杂粮饭</h2>

### 食材（2人份）

- 鸡胸肉 400g
- 番茄 3个
- 西兰花 半颗
- 糙米饭 2盒
- 鸡蛋 2个

### 调味

- 黑胡椒
- 生抽 少量
- 盐 少量

### 制作步骤

1. 糙米和白米1:1煮饭
2. 鸡胸肉切块，加黑胡椒和生抽腌5分钟
3. 少油炒鸡胸肉至变色
4. 加番茄炒出汁
5. 加少量热水焖3分钟
6. 西兰花焯水1分钟
7. 装盒：糙米饭 + 鸡胸肉 + 西兰花 + 水煮蛋

### 保存建议

- 冷藏2天内吃完

<h2 id="lunch-2">🍤 虾仁豆腐盖饭</h2>

### 食材（2人份）

- 冷冻虾仁 300g
- 北豆腐 1盒
- 糙米饭 2盒
- 青菜 1把

### 调味

- 生抽 少量
- 黑胡椒
- 盐 少量

### 制作步骤

1. 虾仁提前解冻
2. 豆腐切块
3. 虾仁少油炒变色
4. 加豆腐
5. 加半碗水焖3分钟
6. 青菜焯水
7. 装盒：糙米饭 + 虾仁豆腐 + 青菜

<h2 id="lunch-3">🍚 肉末豆腐 + 青菜 + 糙米饭</h2>

### 食材（2人份）

- 北豆腐 1盒
- 猪里脊 / 瘦肉 200g
- 上海青 300g
- 糙米饭 2小份

### 调味

- 生抽 1勺
- 黑胡椒
- 少量盐
- 淀粉 半勺（可选）

### 制作步骤

1. 瘦肉切末，加生抽和黑胡椒腌5分钟
2. 豆腐切块焯水1分钟
3. 少油炒瘦肉至变色
4. 加豆腐和半碗热水
5. 小火焖3分钟
6. 上海青快炒1分钟
7. 配糙米饭食用

<h2 id="lunch-4">🍗 鸡腿肉 + 冬瓜汤 + 糙米饭</h2>

### 食材（2人份）

- 去皮鸡腿肉 2块
- 冬瓜 400g
- 糙米饭 2小份

### 调味

#### 鸡腿肉

- 黑胡椒
- 生抽 1勺
- 少量盐

#### 冬瓜汤

- 白胡椒
- 少量盐

### 制作步骤

1. 鸡腿肉去皮后腌10分钟
2. 不粘锅少油煎至两面金黄
3. 冬瓜切块煮10分钟
4. 加白胡椒和少量盐
5. 配糙米饭食用

<h2 id="lunch-5">🐟 清蒸鱼 + 西兰花 + 小米饭</h2>

### 食材（2人份）

- 鲈鱼 1条（约500g）
- 西兰花 半颗
- 胡萝卜 1根
- 小米饭 / 糙米饭 2小份

### 调味

- 生抽 1勺
- 姜丝
- 葱丝
- 白胡椒 少量

### 制作步骤

1. 鱼清洗后加姜丝和白胡椒腌5分钟
2. 水开后蒸8–10分钟
3. 倒掉部分蒸鱼水
4. 加生抽和葱丝
5. 西兰花和胡萝卜焯水
6. 配小米饭食用
