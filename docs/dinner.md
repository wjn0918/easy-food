## 黑胡椒鸡胸肉 + 西兰花 + 玉米

### 食材（2人份）

- 鸡胸肉 400g
- 西兰花 半颗
- 玉米 2根

### 调味

- 黑胡椒
- 生抽 1勺
- 低钠盐 少量

### 制作步骤

1. 鸡胸肉加黑胡椒、生抽、少量盐腌5分钟
2. 空气炸锅180℃烤12–15分钟，中途翻面
3. 西兰花焯水1分钟
4. 玉米蒸10分钟

## 番茄鸡蛋荞麦面 + 生菜

### 食材（2人份）

- 荞麦面 2份
- 番茄 2个
- 鸡蛋 2个
- 生菜 1把

### 调味

- 生抽 少量
- 黑胡椒
- 盐 少量

### 制作步骤

1. 番茄切块，少油炒出汁
2. 加热水
3. 放入荞麦面
4. 打入鸡蛋
5. 最后加入生菜

## 虾仁豆腐 + 黄瓜 + 糙米饭

### 食材（2人份）

- 虾仁 300g
- 北豆腐 1盒
- 黄瓜 2根
- 糙米饭 2小份

### 调味

- 生抽 1勺
- 黑胡椒
- 少量盐

### 制作步骤

1. 虾仁少油炒至变色
2. 加入豆腐块
3. 加半碗水，小火焖3分钟
4. 黄瓜凉拌即可

## 香煎巴沙鱼 + 红薯 + 菠菜

### 食材（2人份）

- 巴沙鱼 2块
- 红薯 2个
- 菠菜 1把

### 调味

- 黑胡椒
- 生抽 1勺
- 低钠盐 少量
- 柠檬汁（可选）

### 制作步骤

1. 巴沙鱼加入调味料腌5分钟
2. 红薯空气炸锅180℃ 20分钟
3. 巴沙鱼中小火每面煎3分钟
4. 菠菜焯水30秒

---

## 菌菇鸡蛋汤 + 凉拌蔬菜 + 玉米/红薯

### 食材（2人份）

- 菌菇 300g
- 鸡蛋 2个
- 生菜/黄瓜 适量
- 玉米2根 或 红薯2个

### 调味

- 白胡椒
- 盐 少量
- 香油几滴（可选）

### 制作步骤

1. 菌菇加水煮5分钟
2. 打入蛋花
3. 加白胡椒和少量盐
4. 生菜或黄瓜凉拌
