# 固定晚餐菜单


---

<h2 id="menu-1">🍗 黑胡椒鸡胸肉 + 西兰花 + 玉米</h2>

> 高蛋白｜低脂｜适合减脂期

### 食材（2人份）

| 食材 | 份量 |
|---|---|
| 鸡胸肉 | 400g |
| 西兰花 | 半颗 |
| 玉米 | 2根 |

### 调味

- 黑胡椒
- 生抽 1勺
- 低钠盐 少量

### 制作步骤

1. 鸡胸肉加黑胡椒、生抽、少量盐腌5分钟
2. 空气炸锅 180℃ 烤 12–15 分钟，中途翻面
3. 西兰花焯水 1 分钟
4. 玉米蒸 10 分钟

<h2 id="menu-2">🍅 番茄鸡蛋荞麦面 + 生菜</h2>

> 热量适中｜饱腹感强｜适合懒人快手

### 食材（2人份）

| 食材 | 份量 |
|---|---|
| 荞麦面 | 2份 |
| 番茄 | 2个 |
| 鸡蛋 | 2个 |
| 生菜 | 1把 |

### 调味

- 生抽 少量
- 黑胡椒
- 盐 少量

### 制作步骤

1. 番茄切块，少油炒出汁
2. 加热水煮开
3. 放入荞麦面
4. 打入鸡蛋
5. 最后加入生菜

<h2 id="menu-3">🍤 虾仁豆腐 + 黄瓜 + 糙米饭</h2>

> 高蛋白｜低负担｜适合晚间恢复

### 食材（2人份）

| 食材 | 份量 |
|---|---|
| 虾仁 | 300g |
| 北豆腐 | 1盒 |
| 黄瓜 | 2根 |
| 糙米饭 | 2小份 |

### 调味

- 生抽 1勺
- 黑胡椒
- 少量盐

### 制作步骤

1. 虾仁少油炒至变色
2. 加入豆腐块
3. 加半碗水，小火焖 3 分钟
4. 黄瓜凉拌即可

<h2 id="menu-4">🐟 香煎巴沙鱼 + 红薯 + 菠菜</h2>

> Omega-3｜低脂｜高饱腹感

### 食材（2人份）

| 食材 | 份量 |
|---|---|
| 巴沙鱼 | 2块 |
| 红薯 | 2个 |
| 菠菜 | 1把 |

### 调味

- 黑胡椒
- 生抽 1勺
- 低钠盐 少量
- 柠檬汁（可选）

### 制作步骤

1. 巴沙鱼加入调味料腌 5 分钟
2. 红薯空气炸锅 180℃ 烤 20 分钟
3. 巴沙鱼中小火每面煎 3 分钟
4. 菠菜焯水 30 秒

<h2 id="menu-5">🍲 菌菇鸡蛋汤 + 凉拌蔬菜 + 玉米/红薯</h2>

> 清淡｜暖胃｜适合晚间轻食

### 食材（2人份）

| 食材 | 份量 |
|---|---|
| 菌菇 | 300g |
| 鸡蛋 | 2个 |
| 生菜/黄瓜 | 适量 |
| 玉米/红薯 | 任选 |

### 调味

- 白胡椒
- 盐 少量
- 香油几滴（可选）

### 制作步骤

1. 菌菇加水煮 5 分钟
2. 打入蛋花
3. 加白胡椒和少量盐
4. 生菜或黄瓜凉拌
